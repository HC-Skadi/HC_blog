# DeepWalk

DeepWalk将graph的每一个节点编码为一个D维向量（Embedding）（无监督学习）

Embedding中隐式包含了graph中的社群、连接、结构信息，可用于后续节点分类（监督学习）





假设相邻的节点之间是相关的
